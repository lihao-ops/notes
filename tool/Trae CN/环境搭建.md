# 环境搭建



>在settings.json中配置maven

```xml
{
    "window.commandCenter": 1,
    "cursor.aipreview.enabled": true,
    "editor.accessibilityPageSize": 16,
    "editor.fontSize": 16,
    "java.configuration.maven.userSettings": "D:\\maven3.9.9\\conf\\settings.xml",
    "java.configuration.maven.globalSettings": "D:\\maven3.9.9\\conf\\settings.xml"
  }
```

此时会使用当前系统环境配置的JDK版本进行编译运行
