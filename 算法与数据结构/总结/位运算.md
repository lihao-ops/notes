# 二进制



## 一、概念

```text
二进制是一种数制，它只包含两个数字：0和1。它是计算机中最基本的数制，因为计算机的基本元件——逻辑门和触发器可以很容易地表示和操作这两个数值。

在二进制数制中，每一位的值仅由0和1组成。数字的每一位都有一个权重，该权重表示该位在数字中的位置。例如，二进制数字1101可以表示成：

1 * 2^3 + 1 * 2^2 + 0 * 2^1 + 1 * 2^0 = 13（十进制）
在这个例子中，二进制数字1101的每一位的权重分别为3、2、1和0。因此，该数字表示的十进制数为13。

在计算机中，二进制数制用于表示各种数据，例如整数、浮点数和字符。在编程中，使用二进制数制可以使代码更简洁、更高效。
```









## 二、运算符

|        |          | Java中二进制运算符列表                                       |                                                              |
| ------ | -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| &      | 按位与   | 对于两个二进制数A和B，按位与运算符将它们逐位相与，并返回一个新的二进制数，其中只有当A和B的对应位都为1时，该位才为1 | 例如，假设A为10110，B为11001，则按位与运算的结果为10000。    |
| ^      | 按位异或 | 对于两个二进制数A和B，按位异或运算符将它们逐位相异或，并返回一个新的二进制数，其中只有当A和B的对应位不同时，该位才为1。 | 例如，假设A为10110，B为11001，则按位异或运算的结果为01111。  |
| \|     | 按位或   | 对于两个二进制数A和B，按位或运算符将它们逐位相或，并返回一个新的二进制数，其中只要A和B的对应位中有一个为1，该位就为1。 | 例如，假设A为10110，B为11001，则按位或运算的结果为11111。    |
| &lt&lt | 左移运算 | 左移运算符（&lt&lt）将数字的二进制表示向左移动指定的位数(左移优先级>右移) | 例如，将8的二进制表示1000向左移动1位，得到10000。将这个二进制数转换为十进制数，得到16。 |
| >>     | 右移运算 | 将一个二进制数向右移动指定的位数，左侧空出的位用0填充。      | 例如，将8的二进制表示1000向右移动1位，得到1100。将这个二进制数转换为十进制数，得到4。 |

二进制运算符在计算机科学和编程中非常常见，因为它们可以用于执行各种逻辑运算和位操作。









## 三、计算



### 除2取余倒序法

>将十进制数转换为二进制数

要将数字转换为二进制，可以使用“除2取余法”。具体步骤如下：

- 将数字除以2，得到商和余数。
- 将余数记录下来，作为二进制位上的数字。
- 将商作为新的数字，重复步骤1和2，直到商为0为止。
- 将得到的二进制位上的数字倒序排列，得到最终的二进制数。



> 以100为例：

1. 100除以2，商为50，余数为0。
2. 将余数0记录下来，作为二进制位上的数字。
3. 将商50作为新的数字，重复步骤1和2，直到商为0为止。
4. 将得到的二进制位上的数字倒序排列，得到最终的二进制数为1100100。
5. 因此，数字100的二进制表示为1100100。

```bash
//100除以2，商为50，余数为0。将余数0记录下来，作为二进制位上的数字。
100/2 = 50	0
//将商50作为新的数字，重复步骤1和2，直到商为0为止。
50/2 = 25	0

25/2 = 12	1

12/2 = 6	0

6/2 = 3		0

3/2 = 1		1

1/2 = 0		1
//将得到的二进制位上的数字倒序排列，得到最终的二进制数为1100100。
1100100
```
